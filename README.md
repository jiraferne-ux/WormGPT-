## 🔧 Manual Setup

```bash
# 1. Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# 2. Pull model (required)
ollama pull godmoded/llama3-lexi-uncensored

# 3. Start Ollama
ollama serve

# 4. Install deps
cd app && npm install && npm run build && cd ..
cd server && npm install && cd ..

# 5. Start server
cd server && node index.js
```
